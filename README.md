# alrale-mock-server
cli

## Install

```shell
$ npm install -g @alrale/mock-server
```

## Usage
``` shell
Usage: cli [options] [command]

Options:
  -v, -version        output the version number
  -h, --help          display help for command

Commands:
  start [options]     启动Mock服务
  design|d [options]  设计器
```
